升级

对ingress玩家来说，没什么比升级更令人兴奋的了。除了可以使用更高级的道具之外，升级还有利于你进行更大范围的探索。尽快升级，上上之选。

1-8级玩家：

由于前8级升级只对AP有要求，所以在日常游玩的同时，请优先考虑建立短连接和多重Field，以增加更多的AP。

  • L1 = 0 AP
  • L2 = 2,500 AP
  • L3 = 20,000 AP
  • L4 = 70,000 AP
  • L5 = 150,000 AP
  • L6 = 300,000 AP
  • L7 = 600,000 AP
  • L8 = 1,2000,000 AP

9级以上玩家：

由于8级之后还需要在勋章上有所突破，所以可能会更加困难。但是升级仍然会带来XM容量的扩充以及充电效率的提高。

  • L 9 = 2,400,000 AP + 1 x + 4 x
  • L10 = 4,000,000 AP + 2 x + 5 x
  • L11 = 6,000,000 AP + 4 x + 6 x
  • L12 = 8,400,000 AP + 6 x + 7 x
  • L13 = 12,000,000 AP + 1 x + 7 x
  • L14 = 17,000,000 AP + 2 x + 7 x
  • L15 = 24,000,000 AP + 3 x + 7 x
  • L16 = 40,000,000 AP + 2 x + 4 x + 7 x
